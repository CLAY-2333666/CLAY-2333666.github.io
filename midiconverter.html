Overwatch MIDI pianist

Upload .mid file:

Start At: The time (seconds) in the song when the script starts reading the MIDI data. Set to values other than 0 to start reading from different parts of the file.
Stop At: The time (seconds) in the song when the script stops reading MIDI data.
Max Voices: Maximum amount of pitches that will be playing at any given time, between 1 and 11. If there are more, they are discarded. Fewer voices may result in better performance.
Max Array Elements: Maximum amount of elements in a single array of the resulting workshop script that contains the song, between 12 and 999. Fewer elements may cause more stuttering, while a high amount of elements may result in too large rule sizes.
