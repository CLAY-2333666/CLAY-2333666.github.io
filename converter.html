Start At: The time (seconds) in the song when the script starts reading the MIDI data. Set to values other than 0 to start reading from different parts of the file.
Stop At: The time (seconds) in the song when the script stops reading MIDI data.
Max Voices: The maximum amount of bots that will be playing, between 1 and 11. The converter automatically determines how many are needed, but will never spawn more than this value. Bots may despawn if the amount of heroes (players + bots) in the custom game exceeds 12.
Max Array Elements: Maximum amount of elements in a single array of the resulting workshop script that contains the song, between 12 and 999. Fewer elements may cause more frequent stuttering, while a high amount of elements may result in too large rule sizes. If unsure, use the default value
Language: Language of the generated workshop script. Choose the same language as your game's text language, otherwise the script can't be imported.
Generate Full Gamemode Settings If unchecked, only the workshop rules containing the song will be generated. Make sure to remove all "Song Data" rules before pasting in the new script.
Make bots invisible Make the bots playing the song invisible. Their primary fire remains visible.
Restrict player slots Restrict the amount of available player slots based on the amount of bots required for the song. Enabling this ensures that bots will never despawn due to joining players.
Choose piano: Select the piano in Paris that the bots will play.
Restrict players' abilities Disable all abilities, ultimates, melee attacks, as well as primary and secondary fires.
